# @umbra/ui

Shared **design tokens** and a small set of headless components so web and
mobile stay visually consistent. Aesthetic: **neobrutalism** — hard edges, bold
borders, high-contrast blocks, chunky shadows, confident type.

Tokens (color, spacing, radius, type scale, shadow) are framework-agnostic and
consumed by Tailwind (web) and NativeWind (mobile).

> Status: package shell only. Implementation begins once the contract + auth
> design is approved.
